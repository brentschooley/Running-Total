Instructions for running RunningTotal:

1. Use NuGet to add the following packages: JSON.NET, WinRTXamlToolkit
2. Add references to the following Infragistics DLLs which are found in the IG folder in the zip file you downloaded:
		InfragisticsWRT.v13.1
		InfragisticsWRT.DataVisualization.v13.1
		InfragisticsWRT.Controls.Charts.XamDataChart.v13.1
		InfragisticsWRT.Controls.Charts.XamGeographicMaps.v13.1 

3. Build the app.  It will work in either Simulator or Local Machine mode.
